<h1>Egalloc - A collage builder </h1>
<h2> Anisha Nagarajan, Chaokang Wu, Gavin Michaels, Peijia Lu, William Tjandra, Zifan Shi </h2>

<p>Egalloc uses the first 30 results from google image search to build a collage. You can also enter some text and it will generate collage of that text. </p>

<p>
The original project was deployed on [here](https://egalloc.herokuapp.com) <br>
The UI updated version by William was deployed on [here](https://e-galloc.herokuapp.com)
</p>
